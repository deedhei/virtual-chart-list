# virtual-chart-list

用于 vue3 虚拟`echarts`图表支持横向和竖向，目前不支持换行

图表目前使用的是`echarts`

支持二次开发

喜欢的打赏小弟小白一下

![](http://39.101.74.144:9090/zxzx/photos/58eb75be30dec1060fb62100dbc77cde.jpg)

## 需要传入的参数

| **选项**           | **描述**                                        | **默认**     | **必填** |
| ------------------ | ----------------------------------------------- | ------------ | -------- |
| **chartData**      | **图表数据**                                    | **无**       | **是**   |
| **itemSizeWidth**  | **图的宽度**                                    | **400**      | **否**   |
| **itemSizeHeight** | **图的高度**                                    | **400**      | **否**   |
| **bufferSize**     | **缓冲数据也就是** **end = start + bufferSize** | **10**       | **否**   |
| **horizontald**    | **方向**                                        | **vertical** | **否**   |

# 使用方法

1、下载

```
npm i virtual-chart-vue3  
```

如果版本有问题就 `npm i virtual-chart-vue3  --legacy-peer-deps`

2、引入

```js
import vueVirtualChart from "virtual-chart-vue3";
const app = createApp(App);
vueVirtualChart(app);
```

3、使用

```js
  <div class="wrapper">
    <vueVirtualChart  :chartData="chartData"></vueVirtualChart>
  </div>
```

注意：

**`wrapper`父亲要给宽高**
